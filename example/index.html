Dark mode

General info

For now it's about styling of the elements

All variables you can find in style/turbo.scss file

If you want change color in dark mode, use these variables with the prefix dark-

Example:

                    
                        $button-background: #ffffff;
                        $dark-button-background: #000000;

Button, selects, modals are not interactive

If you fill input, the label will not be floating

Button

$button-background - background color

$button-font-color - font color

$button-border-color - border color

Inputs

$checkbox-background - checkbox background color

$checkbox-checked-background - checked checkbox background color

$radio-border-color - input type radio border color

$input-left - left padding of all radio and checkbox inputs and toggle button

$toggle-width - width of toggle button
