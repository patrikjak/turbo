General info

For now it's about styling of the elements

All variables you can find in style/turbo.scss file

If you want change color in dark mode, use these variables with the prefix dark-

Example:

                    
                        $button-background: #ffffff;
                        $dark-button-background: #000000;

Button, selects, modals are not interactive

If you fill input, the label will not be floating
